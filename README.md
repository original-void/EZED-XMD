# EZED XMD

A powerful WhatsApp Multi-Device Bot built with Baileys.

## Features

- QR Code on Web
- Multi Device
- Auto Reconnect
- Auto Reply
- Commands
- Fast Response

## Installation

```bash
git clone https://github.com/YOUR_USERNAME/EZED-XMD.git

cd EZED-XMD

npm install

npm start
```

## Deploy

- Push to GitHub.
- Connect the repository to Render.
- Deploy.
- Open your Render URL.
- Scan the QR Code.
- Enjoy EZED XMD.
